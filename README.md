# 8:1 Multiplexer and 1:8 Demultiplexer using Verilog HDL

## Overview

This repository contains the Verilog HDL implementation of an **8:1 Multiplexer** and a **1:8 Demultiplexer**. The project demonstrates the design and implementation of both combinational logic circuits using Verilog HDL.

## Features

- 8:1 Multiplexer
- 1:8 Demultiplexer
- Synthesizable Verilog HDL code
- Separate testbench for functional verification

## Project Structure

```
04-mux-demux-verilog/
├── mux_8to1.v
├── demux_1to8.v
├── tb_mux_8to1.v
├── tb_demux_1to8.v
└── README.md
```

## Tools Used

- Verilog HDL
- Xilinx Vivado

## Learning Objectives

- Understand the working of an 8:1 Multiplexer.
- Understand the working of a 1:8 Demultiplexer.
- Learn the implementation of combinational logic circuits using Verilog HDL.
- Verify the functionality using Verilog testbenches.
